# Tab Navigation
Works with windows and Linux
```
Ctrl + page up/ down
```
## Chrome Shortcuts
```
ctrl + 1
ctrl + 9
ctrl + tab
ctrl + Shift+ o #open bookmark
alt + e #settings menu
ctrl + e #address bar
ctrl + d #for mute unmute microphone
```
